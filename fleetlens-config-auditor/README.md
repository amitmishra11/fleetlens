# fleetlens-config-auditor

The Config Auditor module polls each registered service's Spring Boot Actuator `/actuator/env` endpoint on a configurable interval, diffs the flattened key/value snapshot against the previous capture, persists every snapshot to Postgres (`config_snapshots`), and publishes a `ConfigDriftEvent` on the application event bus whenever keys are added, removed, or modified. It exposes REST endpoints under `/api/v1/config` for listing registered services, fetching the latest or diffed snapshots per service, querying drift events since a timestamp, and building a key-by-environment matrix that highlights configuration values that have diverged across environments. This module requires zero instrumentation of the target service — any Spring Boot app with Actuator's `env` endpoint exposed can be audited. It is packaged as a plain library jar and is wired into the runtime app by `fleetlens-api-gateway`.
